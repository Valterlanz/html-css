Codigo-fonte

Codigo fonte, basicamente deixa as palavras com o mesmo tamanho de fonte e espaço.
Sem fonte = ComPutAdoR
Com fonte = ComPutAdoR
observe que as palavras diminuiram más estão com mesmo tamanho e largura. Para usar o codigo basta colocar <code>.

Identação


num = int(input('Digite um número: '))
if num % 2 == 0:
    print(f'O número {num} é PAR')
else:
    print(f'O número {num} é IMPAR)


Isso é identação, tudo que se escreve em <pre> fica exatamente igual no codigo, como espaços, 
quebra de linha sem o uso de <br>, etc.
